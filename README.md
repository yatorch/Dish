# Dish
Used to power endpoints that will fetch equity and crypto data

**The goal** is for this to be used in simple backtest systems used to test hypotheses for research, since our main goal in terms of exposures is to be exposed to liquid easily marketable assets like equities, etfs, some mutual funds, and cryptocurrencies (both super-liquid stablecoins and alts, and maybe basic features for less-liquid coins found on decentralized exchanges)

To summarize: this should power backtesting systems by obtaining data from simple, trustworthy endpoints, then organizing it, and finally feeding it to basic testing systems used to validate or invalidate basic hypotheses.
